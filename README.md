# Notes App

Простое приложение для создания и управления заметками.

## Установка

```bash
pip install -r requirements.txt
```

## Запуск

```bash
python main.py
```

## Функциональность

- Создание, редактирование и удаление заметок
- Поиск по названию
- Сортировка по дате
- Поддержка светлой и темной темы
